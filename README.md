# Привет, я Вадим 👋

Backend-разработчик на C# и ASP.NET Core. Развиваю коммерческий продукт сервис доставки и практикую PostgreSQL, Docker, CI/CD и production-подходов.

## Чем занимаюсь

- разрабатываю Web API и MVC-приложения на ASP.NET Core;
- проектирую PostgreSQL-схемы и работаю с Entity Framework Core;
- реализую авторизацию, JWT, refresh tokens, OTP и role-based access;
- строю модульный монолит и постепенно выношу бизнес-логику из контроллеров;
- настраиваю Docker Compose, VPS, Nginx, HTTPS и миграции;
- изучаю кеширование, observability, background jobs и надёжные интеграции.

## Основной стек

`C#` `ASP.NET Core` `Entity Framework Core` `PostgreSQL` `Docker` `MVC` `REST API` `JWT` `GitHub Actions`

## Сейчас работаю над

### Доставка-Курорт

Платформа доставки для локальных ресторанов, магазинов и торговых точек.

В проекте есть:

- регистрация и вход по телефону;
- роли покупателя, продавца, курьера и администратора;
- каталог товаров и точек;
- поиск, рейтинги, избранное и рекомендации;
- корзина и оформление заказов;
- раздельная обработка частей заказа для продавцов;
- статусы сборки и доставки;
- Docker Compose и PostgreSQL.
